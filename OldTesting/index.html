Overview

The AnalogIn Module adds analog acquisition capabilities to the Bpod ecosystem.

Features

  • 8 input channels (12-bit)
  • 20kHz+ sampling
  • Input range: -10 to +10V (software selectable to +/- 5 or +/-2.5V and 0 to 10V ranges)
  • MicroSD card to store up to XXX seconds of acquired data
  • USB interface to stream or retrieve data from memory
  • Threshold crossing detection (configurable)
  • 2 UART serial interfaces connect the module to:
    • The state machine
    • The analog output module (direct interface for closed loop applications)
  • I2C interface supported with the I2C module
  • Matlab API
  • Powered by Arduino Due
